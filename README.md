# FrameCraft SaaS

📸 Photography client delivery platform focused on **gallery management, client engagement, analytics, and branded photo delivery workflows.**

---

## Overview
FrameCraft is a full-stack SaaS application built for photographers to simplify how event photos are delivered to clients.

Instead of relying on generic file-sharing tools, photographers can organize galleries, share photos through public links, collect client favorites, monitor engagement, and customize the gallery experience using their own branding.

---

## Core Capabilities

- 📸 Gallery creation and management
- 🖼️ Photo upload and organization
- 🔗 Public gallery sharing through unique links
- ❤️ Client photo favorites
- 📊 Gallery analytics and engagement tracking
- 🎨 Branding and identity customization
- 🔐 Secure authentication and protected access

---

## System Design

```
React + TypeScript Frontend
           ↓
Supabase Authentication
           ↓
Supabase Database + Storage
           ↓
Public Gallery & Analytics Layer
```
